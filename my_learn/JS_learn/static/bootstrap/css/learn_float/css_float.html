1.实验指导书

针对物联网工程专业课程实验内容的特点，设计了物联网综合实验台。 实验台采用模块化设计，包含传感器原理实验区、基础技术实验区、 无线自组网实验区、嵌入式网关实验区、RFID技术实验区、执行控 制技术实验区等；硬件环境支持的实验内容覆盖物联网感知层和网络 层的主要知识体系。

2.物联网综合实验台

在深入理解教执委在物联网专业的规划和多个本专科高校对物联网专 业培养方案的基础上，针对物联网工程专业课程群设计一套有针对性 的实验指导书，例如：物联网通信技术、RFID原理及应用、传感器 原理及应用、传感网原理及应用、物联网控制、物联网工程设计与实 施和嵌入式系统设计等课程都已撰写完毕。

3.物联网教学综合实验平台软件系统

物联网教学综合实验平台软件系统，为用户提供在线式实验指导书、实验 步骤引导、实验报告在线提交和实验报告导出等功能；同时可从数据链路 层获取并展示通信过程中的数据，方便用户对系统进行调试。

<A>自定义指导书
支持对各门课程实验指导书的自定义、维护及管理工作，方便各教学单位收集整理符合 自身教学特色的实验指导书，辅助高校规范实验教学管理过程。
<B>自定义服务端数据处理
为了让学生深切体会服务器端数据处理的过程，实验平台软件系 统提供了数据处理程序自定义功能，学生可以自己编写服务器端 数据处理程序，体会服务器对数据和控制指令的处理过程。






专业性︵面向专业建设︶

产品设计依据教育部计算机教指委编制的《高等学校物联网工程专业实践教学体系与规范（试行）》，针对高校本、专科物联网 工程专业的核心课程基础教学实验，涵盖了物联网工程专业主干课程与硬件相关实验。

先进性

产品以物联网、移动互联网和云计算为理论和技术基础，提出了“口袋实验室”的理念， 使用户可随时随地进行学习、实验和应用开发。

系统行︵面向学科建设︶

本产品以软硬件结合的方式，系统地覆盖了物联网体系结构的感知层、网络层（接入层、汇聚层和交换层），具有完备的物联网底层的 数据链路。各层次均可方便地在本产品上进行实验，用户进行单层或单点实验时，可直接接入本平台的数据链路进行开发、调试，并可 对数据和功能展现，使用户专注与自己的兴趣点之上。

实用性

产品设计目标明确，针对性强，重点突出了本、专科教学实验平台的功能，同时也为基础课程的实验以及科研和研发工作提供支持； 配置灵活，操作简便，在各层次的子层次都配有调试接口软件，方便用户对底层数据的分析和调试。综合试验台模块化设计，集成度高， 可以根据实际需求灵活配置，提高设备利用率。使用综合实验平台可以对各个基础知识点进行教学，同时还可以选择模块上的不同设备 进行自定义综合实验，如智能家居、智能大棚、智能仓库等。相比实验箱、实验沙盘等。可以同时为多个班级（组）提供实验的额实训环境。
